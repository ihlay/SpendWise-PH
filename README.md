# SpendWise PH
 
A Filipino personal budgeting web app built with Django and SQLite.
 
## Features
- Sweldo Tracker (semi-monthly paycheck tracking)
- Utang / Debt Tracker
- Daily Spending Limit Calculator
- Savings Goal Tracker
- Needs vs Wants Tagging
- Most Expensive Day / Category Analytics
- Philippine Holiday Checking (Nager.Date API)
- Progressive Web App (PWA) Support
 
## Tech Stack
- Python / Django
- SQLite
- Bootstrap 5
- Nager.Date Holiday API
- django-pwa
